[https://cli.vuejs.org/zh/guide/](https://cli.vuejs.org/zh/guide/)

## 1.快速原型开发

可以快速识别.vue 文件封装组件插件等功能

```bash
sudo npm install @vue/cli -g
sudo npm install -g @vue/cli-service-global
vue serve App.vue
```
