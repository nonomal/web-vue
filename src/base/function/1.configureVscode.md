# 一、配置 vscode
- 配置插件
  - `Vue 2 Snippets`：ue2的语法高亮，语法提示
  - `Vetur`：vue单文件能够像html，js，css一样编写，注释
  - `wpy-beautify`：vue2单文件格式化
  - `Document This`: 能够一键给代码中的类、函数加上注释，支持函数声明、函数表达式、箭头函数等
