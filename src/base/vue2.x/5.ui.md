## 1.问题集锦

### 1.1 时间选择器

- 表单必填校验失效

  data 中初始化时将初始值设置为数字 0，在 elementui 内部识别为非空值，导致必填校验失效

- 可以输入`e`和`.`

  输入框中可以输入科学记数法`10.1e2`
