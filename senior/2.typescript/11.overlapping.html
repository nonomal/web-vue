# 十一.交叉类型

前言

交叉类型是将多个类型合并为一个类型

interface Person1 {
  handsome: string
}
interface Person2 {
  high: string
}
type P1P2 = Person1 & Person2
let p: P1P2 = { handsome: "帅", high: "高" }
