# 五.组件发布

## 1.npm 官网上注册帐号

npm 官网注册个账号 https://www.npmjs.com/

## 2.工程打包的根目录登陆 npm

```bash
npm login
```

## 3.将打包后的文件发布到 npm 上

```bash
npm publish
```

::: warning 注意
每次发布时，version 都要不一致，所以每次修改都要修改版本
:::
