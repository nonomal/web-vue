# Vue2.x（components）

::: tip 前言
vue 中不同的组件类型：从文件上分为

## 1.sfc

### 1.1 template

::: demo

```vue
<template>
  <div id="main" class="container" style="color: red">
    <p v-if="show">内容 1</p>
    <p v-else>内容 2</p>
  </div>
</template>
<script>
export default {
  data() {
    return {
      show: false,
    }
  },
}
</script>
```

:::

### 1.2 functional

- 组件只用于展示数据，所有动态数据都从父组件传递进来（只有 props），内部没有逻辑交互（无 methods 方法、也没有 mounted 等任何生命周期处理函数），没有状态修改(无 data)，推荐使用函数式组件来提升 vue 的性能

::: demo

```vue
<template functional>
  <ul>
    <li v-for="(item, index) in items" :key="index" @click="itemClick(item)">
      {{ item }}
    </li>
  </ul>
</template>
<script>
export default {
  props: {
    items: {
      type: Array,
      default: () => ["functional"],
    },
    itemClick: {
      type: Function,
      default: () => {},
    },
  },
}
</script>
```

:::

### 1.3 render

::: demo

```js
<script>
export default {
  data() {
    return {
      show: false,
    }
  },
  render: (h) => {
    let childNode
    if (this.show) {
      childNode = h("p", "内容 1")
    } else {
      childNode = h("p", "内容 2")
    }
    return h(
      "div",
      {
        attrs: {
          id: "main",
        },
        class: {
          container: true,
        },
        style: {
          color: "red",
        },
      },
      [childNode]
    )
  },
}
</script>
```

:::

### 1.4 abstract

#### 1.4.1 keepAlive

- 路由中

```js
{
  path: 'list',
  name: 'itemList', // 列表页
  component (resolve) {
    require(['@/pages/item/list'], resolve)
 },
 meta: {
  keepAlive: true,
  title: '列表页'
 }
}
```

- 组件中

```vue
<template>
  <div id="app" class="wrapper">
    <keep-alive>
      <!-- 需要缓存的视图组件 -->
      <router-view v-if="$route.meta.keepAlive"></router-view>
    </keep-alive>
    <!-- 不需要缓存的视图组件 -->
    <router-view v-if="!$route.meta.keepAlive"></router-view>
  </div>
</template>
```

- 生命周期
  - activated
  - deactivated

#### 1.4.2 component

- 动态组件切换不同类型的组件

::: demo

```vue
<template>
  <div>
    <el-form>
      <el-form-item>
        <component
          v-if="data.name"
          :is="data.name"
          v-bind="data.props"
        ></component>
      </el-form-item>
    </el-form>
    <web-button @click="handleChange('el-input')">el-input</web-button>
    <web-button @click="handleChange('el-radio')">el-radio</web-button>
    <web-button @click="handleChange('el-select')">el-select</web-button>
  </div>
</template>
<script>
export default {
  data() {
    return {
      data: {
        name: "",
        props: {},
      },
      target: Object.freeze({
        "el-input": {
          name: "el-input",
          props: { value: "10" },
        },
        "el-radio": {
          name: "el-radio",
          props: { value: "1", label: "1" },
        },
        "el-select": {
          name: "el-select",
          props: { value: "相机" },
        },
      }),
    }
  },
  methods: {
    handleChange(name) {
      this.data = this.target[name]
    },
  },
}
</script>
```

## 3.extend

- 场景:vue 组件中有些需要将一些元素挂载到元素上,这个时候 extend 就起到作用了

```js
var Profile = Vue.extend({
  template: "<p>{{extendData}}</br>实例传入的数据为:{{propsExtend}}</p>", //template对应的标签最外层必须只有一个标签
  data: function () {
    return {
      extendData: "这是extend扩展的数据",
    }
  },
  props: ["propsExtend"],
})

// 创建的构造器可以挂载到元素上,也可以通过 components 或 Vue.component()注册使用

// 挂载到一个元素上。可以通过propsData传参.

new Profile({ propsData: { propsExtend: "我是实例传入的数据" } }).$mount(
  "#app-extend"
)
```

- 场景 2：

```js
export default Vue.extend({
  props: {},
  template: `<div>子组件</div>`,
})
```

```vue
<template>
  <div>
    <div id="parent"></div>
  </div>
</template>
<script>
import child from "./child"
export default {
  name: "parent",
  component: {
    child,
  },
  mounted() {
    new child({
      props: {},
      methods: {},
    }).$mount("#parent")
  },
}
</script>
```

## 4.$createElement

```vue
<template>
  <div>
    <div id="parent"></div>
  </div>
</template>
<script>
import child from "./child"
export default {
  name: "parent",
  component: {
    child,
  },
  mounted() {
    const h = this.$createElement
    let vNode = h(child, {
      props: {},
      methods: {},
    })
    this.$confirm("提示", {
      title: "提示",
      message: vNode,
      showCancelButton: true,
      confirmButtonText: "确定",
      concelButtonText: "取消",
      type: "warning",
    })
  },
}
</script>
```

## 5.patch

```vue
<template>
  <div>
    <div id="parent"></div>
  </div>
</template>
<script>
import child from "./child"
export default {
  name: "parent",
  component: {
    child,
  },
  mounted() {
    const h = this.$createElement
    let vNode = h(child, {
      props: {},
      methods: {},
    })
    const dom = document.getElementById("parent")
    this.__patch__(dom, vNode)
  },
}
</script>
```
