# 二.Vue2.x（keepAlive）

::: tip 前言
vue 项目开发中 keepAlive 缓存组件到内存中，优化页面性能
:::

## 1.用法

- 路由中

```js
{
  path: 'list',
  name: 'itemList', // 列表页
  component (resolve) {
    require(['@/pages/item/list'], resolve)
 },
 meta: {
  keepAlive: true,
  title: '列表页'
 }
}
```

- 组件中

```vue
<div id="app" class="wrapper">
    <keep-alive>
        <!-- 需要缓存的视图组件 -->
        <router-view v-if="$route.meta.keepAlive"></router-view>
     </keep-alive>
      <!-- 不需要缓存的视图组件 -->
     <router-view v-if="!$route.meta.keepAlive"></router-view>
</div>
```
